# 의존성 목록

이 목록은 asdf 버전 0.16.0 이상을 위한 것입니다. 이전 버전의 asdf는 추가적인 의존성이 필요합니다.

asdf 자체는 다음이 설치되어 있어야 합니다:

* Bash 버전 `3.2.48`
* Git 버전 `1.7.7.2`

::: tip 노트

asdf 플러그인은 사용하기 전에 추가적인 프로그램의 설치가 필요할 수 있습니다. 플러그인을 설치하기 전에 플러그인의 문서를 읽고 해당 플러그인의 모든 의존성이 설치되어 있는지 확인하세요.

:::
